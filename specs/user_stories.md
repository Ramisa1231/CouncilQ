# User Stories

## Waste And Recycling MVP

- As a resident, I want to know which bin to use for an item so I can dispose of it correctly.
- As a resident, I want to check my bin collection day so I know when to put bins out.
- As a resident, I want to report a missed bin so I can resolve a collection issue.
- As a resident, I want to understand hard waste collection options so I can book or prepare correctly.
- As a resident, I want to know where to dispose of hazardous or electronic waste safely.

## Safety And Trust

- As a user, I want answers to include sources so I can verify them.
- As a user, I want the assistant to ask clarifying questions instead of guessing.
- As a project owner, I want the assistant to ignore prompt injection in retrieved pages.
- As a project owner, I want PII protected in logs and tool calls.

## Council Documents

- As a resident, I want answers from council policies and by-laws to cite the exact document and page so I can verify the detail.
- As a project owner, I want CouncilQ to download and extract official PDFs once so repeated questions do not reprocess the same documents.
- As a developer, I want document ingestion to be testable from local fixtures without relying on the live City of Adelaide website.
